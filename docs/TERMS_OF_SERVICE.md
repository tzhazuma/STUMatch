# UniMatch（STUMatch）用户服务协议

**最后更新日期：2026-07-15**

欢迎使用 UniMatch（又称 STUMatch，以下简称“本平台”）。本平台由校园团队运营，面向在校学生及校友提供学术、日常、恋爱交友等匹配与社区服务。请你（以下简称“用户”）在注册、登录或使用本平台前仔细阅读本协议。一旦你完成注册或使用本平台服务，即视为你已充分理解并同意受本协议全部条款的约束。

## 1. 服务范围

1.1 本平台为用户提供基于校园身份的资料展示、兴趣匹配、问卷匹配、好友请求、即时消息、留言板、活动推荐及相关的社区服务（以下简称“服务”）。

1.2 平台可能根据运营需要随时调整、升级、暂停或终止部分或全部服务，并会尽可能提前通过公告或站内通知告知用户。

## 2. 账号注册与管理

2.1 用户须使用真实、有效、可验证的校园邮箱或手机号进行注册，并保证所提供的资料真实、准确、完整。

2.2 用户注册成功后获得本平台账号的使用权，账号所有权归本平台所有。用户应妥善保管账号、密码及验证码，对账号下的一切行为承担法律责任。

2.3 用户不得转让、出租、借用或售卖账号，亦不得冒充他人、伪造身份注册或使用本平台。

2.4 如发现账号被盗用、被冒用或存在安全风险，用户应立即通知平台并采取相应措施。

## 3. 用户信息与隐私保护

3.1 平台重视用户隐私，具体信息收集、使用、存储、共享及权利行使方式请参见《隐私政策》。

3.2 用户在注册时勾选“我已阅读并同意服务协议和隐私政策”，即表示同意平台按照本协议及《隐私政策》处理相关信息。

## 4. 用户行为规范

4.1 用户在使用本平台时，应遵守中华人民共和国法律法规、社会公德及校园管理规定，不得发布、传播或以任何方式使用含有以下内容的信息：

- 反对宪法所确定的基本原则的；
- 危害国家安全、泄露国家秘密、颠覆国家政权、破坏国家统一的；
- 损害国家荣誉和利益的；
- 煽动民族仇恨、民族歧视，破坏民族团结的；
- 侮辱、诽谤、威胁他人，侵犯他人合法权益的；
- 散布淫秽、色情、赌博、暴力、凶杀、恐怖或者教唆犯罪的；
- 散布谣言、虚假信息，扰乱社会秩序的；
- 侵犯他人知识产权、商业秘密、肖像权、隐私权等合法权益的；
- 含有广告、垃圾信息、诈骗信息或恶意链接的；
- 其他违反法律法规或平台规则的内容。

4.2 用户不得利用本平台从事以下行为：

- 恶意注册、批量注册账号；
- 使用自动化脚本、爬虫或其他技术手段非法获取平台数据；
- 对他人进行骚扰、 stalking、人肉搜索或线下滋扰；
- 诱导他人脱离平台进行交易、转账或线下危险会面；
- 传播计算机病毒、木马或其他恶意程序；
- 试图绕过平台安全机制或进行渗透测试。

4.3 用户发布的内容由用户本人承担责任。平台有权根据法律法规、本协议及社区规则对内容进行审核、删除、屏蔽或限制传播。

## 5. 匹配与社区安全

5.1 平台提供的匹配结果基于用户填写的资料、问卷及算法模型生成，仅供参考。用户应自行判断并谨慎决定是否与他人建立联系或线下见面。

5.2 用户线下见面应遵循安全、公开、自愿的原则，建议选择公共场所并告知亲友。平台不对用户线下行为及后果承担责任，但会配合有关部门处理违法违规行为。

5.3 平台鼓励用户在发现可疑账号、骚扰、诈骗或其他违规行为时，通过举报功能或联系客服进行反馈。
