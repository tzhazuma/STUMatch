# UniMatch（STUMatch）隐私政策

**最后更新日期：2026-07-15**

UniMatch（又称 STUMatch，以下简称“本平台”或“我们”）非常重视用户的隐私和个人信息保护。本政策旨在向你说明我们如何收集、使用、存储、共享、保护你的个人信息，以及你享有的相关权利。请你仔细阅读并充分理解本政策。一旦你使用本平台服务，即视为同意本政策的全部内容。

## 1. 我们收集的信息

1.1 **账号信息**：包括你注册时提供的邮箱、手机号、昵称、学校、密码及验证码信息。

1.2 **个人资料**：包括性别、出生日期、年龄、学历、专业、MBTI、兴趣爱好、现居地、个人简介、研究方向、交友目的、家庭与未来规划等。你可以选择填写或跳过部分项目。

1.3 **问卷回答**：包括你在基础资料、学术交流、日常生活、恋爱交友等问卷中的选择与文本回答。

1.4 **使用数据**：包括你的登录记录、浏览记录、匹配记录、反馈记录（喜欢/不喜欢/跳过）、好友关系、聊天记录、留言板发布内容及举报记录。

1.5 **设备与日志信息**：包括设备型号、操作系统版本、IP 地址、浏览器类型、访问时间、错误日志及操作行为数据。

1.6 **位置信息**：你填写的现居地或学校位置信息，平台不会主动持续获取你的实时地理位置，除非你在特定功能中单独授权。

1.7 **图片与头像**：你主动上传的头像或其他图片内容。

## 2. 我们如何使用信息

2.1 **提供服务**：用于账号注册与登录、资料展示、匹配推荐、好友与聊天、问卷分析、消息通知及社区功能。

2.2 **改进体验**：用于分析用户行为、优化匹配算法、改进产品功能、修复问题及开展用户研究。

2.3 **安全保障**：用于身份验证、反欺诈、反垃圾、内容审核、风险识别及保护账号安全。

2.4 **合规与沟通**：用于履行法律法规义务、响应你的请求、发送服务通知及处理投诉与举报。

2.5 **个性化推荐**：在获得你同意的前提下，基于你的资料、兴趣及行为数据向你推荐可能感兴趣的用户、活动或内容。

## 3. 信息的存储与保护

3.1 我们采用合理的技术与管理措施保护你的个人信息，防止数据泄露、篡改、丢失或被未经授权访问。

3.2 你的数据存储在中华人民共和国境内或经你同意的境外服务器上，存储期限为实现本政策所述目的所必要的最短时间，除非法律法规另有要求。

3.3 我们将采取加密、脱敏、访问控制、审计日志等手段提升数据安全性，但请你理解，任何网络传输与存储均无法做到绝对安全。

## 4. 信息的共享与披露

4.1 我们不会向第三方出售你的个人信息。

4.2 在以下情形中，我们可能会共享或披露你的信息：

- **获得你的明确同意**；
- **服务提供商**：委托可信赖的技术服务提供商进行服务器托管、短信发送、邮件推送、数据分析、内容审核等；
- **法律要求**：根据法律法规、司法程序、行政机关要求或公共安全需要而披露；
- **合并与收购**：如发生平台合并、分立、收购或资产转让，我们将通知你并确保接收方继续遵守不低于本政策标准的个人信息保护义务；
- **保护权益**：为保护平台、用户或公众的合法权益而必要披露时。

4.3 我们将对第三方接收方的数据使用进行约束，并要求其采取相应安全措施。

## 5. Cookie 与类似技术

5.1 为了提升用户体验，我们可能在 Web 端使用 Cookie、LocalStorage 或类似技术，用于记录登录状态、偏好设置及防止欺诈。

5.2 你可以通过浏览器设置管理或清除 Cookie，但部分功能可能因此受到影响。

## 6. 你的权利

6.1 你有权访问、更正、删除你的个人信息，有权撤回部分授权，有权注销账号。

6.2 你可以通过平台“个人中心—设置”或联系客服行使上述权利。我们将在合理期限内响应你的请求，除非法律法规另有规定。

6.3 账号注销后，我们将依据法律法规删除或匿名化你的个人信息，但法律法规要求保留的信息除外。

## 7. 未成年人保护

7.1 本平台主要面向高等院校在校学生。我们重视未成年人的个人信息保护。

7.2 未满 18 周岁的用户应在监护人指导下使用本平台，并在注册时如实提供年龄信息。我们将根据年龄信息采取相应的保护措施，限制部分功能的使用。

7.3 如果我们发现未成年人在未获得监护人同意的情况下提供了个人信息，我们将尽快删除相关信息。

## 8. 第三方链接与服务

8.1 本平台可能包含第三方网站、服务或内容的链接。本政策不适用于第三方服务，请你在使用前查阅相关第三方的隐私政策。

## 9. 政策更新

9.1 我们可能会根据法律法规变化或服务调整不时更新本政策，并在平台显著位置公告或通知你。

9.2 如更新内容涉及你重大权益，我们将通过弹窗、邮件或短信等方式提醒你。继续使用本平台服务视为同意更新后的政策。

## 10. 联系我们

如对本隐私政策或个人信息处理有任何疑问、意见或投诉，可通过以下方式联系我们：

- 客服邮箱：privacy@unimatch.example.com
- 反馈入口：App / Web 内“设置—意见反馈”
